# Slugbook Recipe Manager
Contributers: Emma Nguyen, Nathan Dong, Ann Sasi, Chris Hein, Jennie Lin, Aryan Sidbatte
Date: June 14, 2025

# Overview
Slugbook Recipe Manager is a web-based application built with py4web, Vue,js, and SQL-based databases. This recipe manager allows users to create, browse, organize, share and manage recipes, such as ingredient management and calorie calculation. The app also inetgrates and imports data from [TheMealDB](https://www.themealdb.com/) API for initial recipe population.

# Features
- **User account creation and login system.**
- **Secure recipe editing logic: only allow authors to modify their own recipes.**
- **Ability to search ingredients by name.**
- **Ability to add new ingredients.**
- **Ability to search recipes by name and type.**
- **Ability to create new recipes.**
- **Automatically compute total calories per recipe based on ingredients and quantities.**
- **Integration with TheMealDB API.**
- **Public JSON API for ingredient and recipe searching.**
- **Automatically scale ingredients when changing number of servings.**
